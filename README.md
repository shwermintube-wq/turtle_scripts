# turtle_scripts

***WORK IN PROGRESS***


Turtle Scripts for Computer Craft

To Install the Scripts:

On a computer in minecraft,

Make sure you have a lib directory,

Inside your root directory

Type edit startup

Inside startup.lua, type shell.run(main)

Press ctrl then enter on save

Press ctrl then enter on exit to leave the editor

Type mkdir lib if you don't already have one

Type wget https://raw.githubusercontent.com/shwermintube-wq/turtle_scripts/refs/heads/main . to pull the repository into your root directory
